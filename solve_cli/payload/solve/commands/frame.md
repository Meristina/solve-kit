# /solve.frame — Phase 0 (Commander)

**Constitution check:** `.solve/memory/constitution.md` (Art. III mirror-language,
Art. VI minimal MECE). `$MISSION` = `$ARGUMENTS`.

## Do
1. **Read in**: `$MISSION/dossier.md` (the one-line problem).
2. **Restate** the problem in one sentence and **detect the sector/context**.
3. **Ask 2–3 questions that change the plan** (scope, desired outcome, constraints,
   stakeholders, data available, deadline) — each with a recommended default. **Wait
   for answers** unless the user says "go".
4. **Write out**: append to `$MISSION/dossier.md` → set `sector`, record the
   restated problem, the answers/defaults as assumptions (tagged), and the
   one-line justification of the minimal phase set you'll run (Art. VI).

## Done when
`$MISSION/dossier.md` has the sector and the framing assumptions, and the plan-set is
chosen. No artifact file of its own — framing lives in the Dossier. Mirror the user's
language.
